WS3 운영 콘솔

Dev Preview 단계입니다. 친구/지인 테스트용 가벼운 초대코드 게이트입니다. 실 worker action 은 별도 Invoke Token + server-side guard 로 추가 보호되며, Cloudflare Access 는 production-grade 운영 단계에서 재검토합니다.

상태: 대기
준비 개발 개방 모드 수동 운영 자동 알림 꺼짐
메인 거래소: 빗썸  · 업비트 보조 검증  · 바이낸스 참고
호출 토큰 입력 없음 · 엔드포인트 자동 설정 · 최종 보안 강화(서버측 토큰 관리·접근 게이트)는 공개 배포 전 별도 단계.

9. 테스트 알림 발송

⚠️ 테스트 전송(TEST_ONLY). 실 운영 알람 / 매수 추천 / 수익 보장 아님. 1건만 발송하고 60초 중복 방지 가드가 동작합니다. 서버 측 `WS3_CANDIDATE_TEST_ENABLED='true'` 필수. 확인 문구 정확히 일치 + 선택 후보 필요.
